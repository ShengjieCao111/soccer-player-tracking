# 足球球员多目标跟踪与运动分析
本项目为计算机视觉课程实训项目，基于Python与OpenCV实现足球比赛视频内球员多目标跟踪及运动数据自动化分析。

## 项目简介
项目集成五大核心功能：球员目标跟踪、运动轨迹提取、跑动速度统计、场地中线越界检测、球员活动热力图可视化，完整实现视频输入至运动分析结果输出全流程，是计算机视觉算法在体育赛事数据分析领域的轻量化落地实践。

## 项目背景与意义
随着计算机视觉技术持续发展，体育视频智能分析成为热门应用方向。传统人工统计球员跑动数据存在效率低下、误差较大等缺陷，本项目依靠自动跟踪算法快速提取球员跑动速度、活动范围、越界时长等量化数据，能够为球队战术复盘、球员表现分析提供客观数据支撑。
本次实训以足球赛事视频作为处理对象，基于传统视觉跟踪算法完成整套运动分析逻辑，完整掌握视频目标检测跟踪、轨迹计算、数据可视化开发流程。

## 核心功能实现流程
1. 视频帧读取与预处理：逐帧解析比赛视频，完成画面尺寸统一、灰度降噪预处理；
2. 球员多目标跟踪：使用传统视觉跟踪算法识别场内球员，分配独立ID持续跟踪，避免目标丢失与ID切换；
3. 运动数据计算：基于连续帧坐标计算球员实时跑动速度，判断球员是否跨越场地中线并记录时长；
4. 轨迹与热力图生成：保存球员全程运动坐标，使用matplotlib绘制全场活动热力图；
5. 结果输出：生成标注跟踪轨迹的视频、热力图片、结构化统计CSV文件。

## 项目运行步骤
1. 安装全部Python第三方依赖库；
2. 将足球比赛视频文件放入input目录，修改代码内视频路径参数；
3. 打开main.ipynb完整执行代码块；
4. 程序逐帧处理视频并实时展示跟踪画面；
5. 处理完成后在output文件夹查看轨迹图、热力图与统计数据表。

## 项目难点与优化方案
1. 球员遮挡ID切换问题
难点：球员互相遮挡、与球门/边线重叠时跟踪目标丢失、ID错乱
方案：结合目标尺寸、色彩特征做前后帧匹配，降低ID跳变概率
2. 场地坐标换算误差
难点：像素坐标无法直接换算真实跑动距离，速度计算存在偏差
方案：基于场地标准长宽建立像素-实际距离映射比例，修正速度计算逻辑
3. 热力图可视化效果模糊
难点：大量轨迹点叠加后画面层次不清晰
方案：调整热力图权重衰减参数，区分高频、低频活动区域
